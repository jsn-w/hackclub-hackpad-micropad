# Parts Used
- 1x XIAO RP2040
- 1x 3D printed case: base
- 1x 3D printed case: cap
- 3x SK6812MINI LEDs
- 6x Cherry MX Switches
- 6x DSA Keycaps
- 1x 0.91" 128x32 OLED Display
